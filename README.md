# wen
CLI natural language date parser in rust


### Options:
- `--unix` to output just unix timestamp

### Examples:

```
> wen now 

Unix Time: 1756398746

                | C-time                    | ISO 8601                  | RFC 2822
----------------+---------------------------+---------------------------+---------------------------------
  Local         | Thu Aug 28 09:32:26 2025  | 2025-08-28T09:32:26-07:00 | Thu, 28 Aug 2025 09:32:26 -0700
  UTC           | Thu Aug 28 16:32:26 2025  | 2025-08-28T16:32:26+00:00 | Thu, 28 Aug 2025 16:32:26 +0000
```
